# voicevue-ai-v2

VoiceVue AI v2 — voice interview platform with Prisma and Next.js App Router.

## Stack

Next.js, Prisma, TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/voicevue-ai-v2.git
cd voicevue-ai-v2
npm install
npm run dev   # or: npm start
```
